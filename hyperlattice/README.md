# HyperLattice - City Intelligence Product

## Structure
```
hyperlattice/
├── frontend/      - TypeScript/React
├── ingestion/     - Elixir/Phoenix
├── simulation/    - Python Ray workers
└── intervention/  - Python intervention engine
```

## Data Ingestion
1. Open data ETL
2. Real-time IoT/5G
3. Social/health streams

## Simulation
- Ray worker dispatch
- SDE ensembles: traffic, energy, water, health, emergency, env, social

## Intervention Engine
1. Counterfactual simulation
2. Equity check
3. OCaml gate validation
4. Valid/Uncertain/Invalid handling
